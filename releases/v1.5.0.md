# Release v1.5.0

**Release date:** 2026-07-06

## Highlights

- Added a Music Recommendation block for album and track recommendations.
- Added format and media-type filtering to the Media Cover Grid block.
- Hardened visual link preview loading and cache behavior.
- Refactored media recommendation editors around shared search and preview helpers.
- Added documentation for media block filter settings.

## Features

- **Music Recommendation:** New block with MusicBrainz/Cover Art Archive-powered search, cover preview, frontend playback controls, and media-cover-grid integration.
- **Media Cover Grid:** New frontend filter controls for item format and media type, plus expanded normalizers for music and additional media metadata.
- **Shared Media Editors:** Reused search state, feedback, result list, and preview components across media, music, book, and videogame recommendation editing flows.
- **Visual Link Preview:** Safer preview fetch handling, cached-read-first behavior, and reduced DNS lookup work before serving cached previews.
- **Docs:** README coverage for the new media filtering options.

## Files Changed

- `blocks/music-recommendation/` - New Gutenberg block, editor UI, frontend view script, styles, and renderer.
- `inc/music-recommendation.php` - Server-side music lookup, metadata normalization, and block support.
- `blocks/media-cover-grid/` - Filter attributes, frontend filter controls, rendering updates, and styling.
- `inc/media-cover-grid-normalizers.php` - Normalization support for music and additional media filter metadata.
- `blocks/shared/media/` - Shared editor search state and results components.
- `blocks/media-recommendation/`, `blocks/videogame-recommendation/`, `blocks/book-rating/` - Editor refactors using shared media helpers.
- `blocks/visual-link-preview/render.php` and `inc/visual-link-preview-async.php` - Hardened preview loading and cache behavior.
- `README.md` - Documented media filter settings.

## Breaking Changes

None. Fully backward-compatible with previous versions.

## Notes

- Existing media recommendation blocks continue to render without migration.
- The new Music Recommendation block is opt-in and registered alongside the existing block set.
- Recommended update for the new music block, media cover filtering, and link preview hardening.
