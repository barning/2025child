# Release v1.4.0

**Release date:** 2026-06-04

## Highlights

- Added dynamic block discovery and metadata documentation for developers.
- Refactored media-cover-grid module structure for better maintainability.
- Enhanced RSS footer with custom messages and link management.
- Auto-populated media-cover-grid page with improved deduplication.
- Improved post-likes block with selectable emoji and customizable CTA text.
- Updated GitHub Actions workflow for modern release process.
- Node.js 24 compatibility and dependency updates.

## Features

- **Dynamic Blocks:** Automatic block metadata discovery and developer documentation.
- **Media Cover Grid:** Auto-populated page, stronger deduplication, refactored helpers.
- **RSS Footer:** Custom messages via post editor metabox, footer link management, improved hook compatibility.
- **Post Likes:** Selectable emoji options, customizable CTA text, improved pill layout.
- **Build & CI:** Refactored GitHub Actions workflows, Node.js 24 support, .nvmrc added.

## Files Changed

- `blocks/media-cover-grid/` — Auto-populated pages, deduplication improvements.
- `blocks/post-likes/` — Selectable emoji, CTA customization, layout improvements.
- `inc/media-cover-grid.php` — Refactored helper functions, cleaner module structure.
- `inc/rss-feed-footer.php` — Custom message support, link management improvements.
- `.github/workflows/` — Updated for modern Node.js and release process.
- `package.json` — Dependencies updated for Node 24 compatibility.

## Breaking Changes

None. Fully backward-compatible with previous versions.

## Notes

- No data migration required.
- All features are opt-in through block settings.
- Recommended update for improved development experience and bug fixes.
