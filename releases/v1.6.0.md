# Release v1.6.0

**Release date:** 2026-07-11

## Highlights

- Added portrait videogame cover support through optional SteamGridDB integration.
- Added selectable cover variants to the Videogame Recommendation editor.
- Updated Media Cover Grid rendering to support portrait, square, and landscape cover dimensions.
- Improved Media Cover Grid filter controls with horizontal scrolling and better mobile behavior.
- Removed the Media Cover Grid max-items limit so all matching items can render.

## Features

- **Videogame Recommendation:** RAWG search results can now be enriched with SteamGridDB portrait covers when a SteamGridDB API key is configured.
- **Cover Variants:** Editors can choose among available SteamGridDB cover variants after selecting a game.
- **Media Cover Grid:** Cover output now includes format-specific dimensions, lazy-loading hints, and rendering support for multiple cover shapes.
- **Filter Controls:** Type filters use a horizontally scrollable control surface that behaves better with more media types and small screens.

## Files Changed

- `inc/videogame-recommendation.php` - Added SteamGridDB API key handling, API requests, cover normalization, and RAWG search enrichment.
- `blocks/videogame-recommendation/` - Added cover format and cover variant attributes, editor picker UI, preview adjustments, and documentation updates.
- `blocks/media-cover-grid/` - Updated render output, filter control markup, frontend filtering, and responsive styles for mixed cover formats.
- `inc/media-cover-grid-normalizers.php` - Updated media cover normalization for cover format handling.

## Breaking Changes

None. Fully backward-compatible with previous versions.

## Notes

- SteamGridDB support is optional. Existing RAWG-only videogame recommendations continue to work.
- Existing Media Cover Grid blocks render without migration.
- Recommended update for sites using videogame recommendations or large mixed-media cover grids.
